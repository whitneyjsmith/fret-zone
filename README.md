# FretZone

Guitar Pro tab editor for web, built with React.

Originally a project to practice [Thinking in React](https://reactjs.org/docs/thinking-in-react.html).

## Pre-requisites

- Node.js 25+
- Yarn 1.22.x

## To run locally

Install local dependencies

`yarn install`

Run the app

`yarn start`

## To run tests

`yarn test`
